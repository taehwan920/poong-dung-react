# Poong-dung-react
# 
# 
## 한강 수온 체크 웹앱 Poong-dung 입니다.
# 
# 
> ### 프로젝트 설명
>   > #### 한강 수온 및 수온 통계 확인이 가능합니다.
# 
# 
> ### 저는 이 프로젝트를 통해...
# 
>   > #### Open API를 활용하여 데이터를 추출하여 활용하는 방법을 안다.
>   > #### RDB를 활용할 줄 안다.
>   > #### 파이썬을 통한 데이터 가공 방법을 안다.
>   > #### 스케줄러 기능을 구현할 줄 안다.
>   > #### Node.js/Express.js를 이용해 RESTful API를 구현할 줄 안다.
>   > #### React js를 활용하여 클라이언트 단을 구현할 줄 안다.
